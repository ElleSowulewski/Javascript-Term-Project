This is a HTML5 Canvas & Javascript game based off of
the classic game Breakout. 

Play by using the A/Left-Arrow key to move the paddle
left and the D/Right-Arrow key to move the paddle right.
Break all the blocks to advance to the next level which
will challenge you with faster ball movement.
